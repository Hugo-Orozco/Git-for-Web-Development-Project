Questions

1. What is Semantic HTML?
  - Semantic HTML are tags that are clearly defined like table, which clearly means table, and div is not, because you don't know what that means.
2. What is HTML used for?
  - HTML is used to display content into a browser from the main index.html file.
3. What is an attribute and where do we put it?
  - An attribute is a property that you can pass on an HTML tag.
4. What is the h1 tag used for? How many times should I use it on a page?
  - h1 tag is the largest heading, it will display the largest text. You should only really be using this tag once.
5. Name two tags that have required attributes
  - Tag input and select.
6. What do we put in the head of our HTML document?
  - You pass in the main file tags such at meta, link, style, script, and title.
7. What is an id?
  - An id is an attribute that can only be used once, since it's a unique identifier.
8. What elements can I add an id to?
  - You can add an id to pretty much any element, such as a div, selector, p, a, etc...
9. How many times can I use the same id on a page?
  - You can only use an id on your HTML document once.
10. What is a class?
  - A class is an attribute that you can use more than once and it defined certain elements that have it assigned.
11. What elements can I add a class to?
  - Just like an id, you can pretty much add a class on any tag.
12. How many times can I use the same class on a page?
  - You can use a class as many times as you please.
13. How do I get my link to open in a new tab?
  - You want to use the anchor <a> tag and pass in the attribute href='link'.
14. What is the alt attribute in the image tag used for?
  - The alt attribute is additional information that defines the <a> tag, depending on the type of content from the url.
15. How do I reference an id?
  - You reference an id in CSS by calling the # symbol and then the id text, for example #customId.
16. What is the difference between a section and a div
  - The contents on a section are grouped. A div doesn't have a special meaning and represents children elements.
17. What is CSS used for?
  - CSS is used to style HTML and make it look pretty.
18. How to we select an element? Example - every h2 on the page
  - You select an element in CSS by passing h2 {}.
19. What is the difference between a class and an id? - Give me an example of when I might use each one
  - A class you can use more than once, and id you can only use once. It you want to refer to an element and style it alone then you can use an id, since it will be unique, but if you want to add the same style to multiple elements then using a class would be more ideal, since you can assign it to more than one element.
20. How do we select classes in CSS?
  - You select classes in CSS by first typing in a . then the class name, for example .CustomClassName {}.
21. How do we select a p element with a single class of “human””?
  - You would first call the class, then the element, example .human p {}
22. What is a parent child selector? When would this be useful?
  - It can be used to style a certain element that's inside of another element. It would be useful if you were trying to call every <p> in a <div> for the entire document.
23. How do you select all links within a div with the class of sidebar?
  - You can call it by using .sidebar div {}.
24. What is a pseudo selector?
  - It is a special state in which an element might be in, for example the you could hover an <a> tag, you would use it in CSS like, a:hover. You can get more specific in you want as well.
25. What do we use the change the spacing between lines?
  - You usually use the <br> tag which stands for break line, but you can also use styling in CSS.
26. What do we use to change the spacing between letters?
  - You use the letter-spacing property in CSS.
27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
  - You pass the property { text-transform: uppercase; }, { text-transform: lowercase; }, and { text-transform: capitalize; }.
28. How do I add a 1px border around my div that is dotted and black?
  - You pass the property { border: 1px dotted black; } .
29. How do I select everything on the page?
  - You use the universal symbol * to select everything globally.
30. How do I write a comment in CSS?
  - You surround text with /* */.
31. How do I find out what file I am in, when I am using the command line?
  - pwd
32. Using the command line - how do I see a list of files/folders in my current folder?
  - ls
33. How do I remove a file via the command line? Why do I have to be careful with this?
  - You use rm to delete a file and rm -rf to delete an entire directory. You meed to be careful with this because you could accidentally delete important files or folders by accident.
34. Why should I use version control?
  - Because if you push an update that breaks everything and nothing works, then you could always go back to a previous version that worked, until you push a newer update that fixes the previous issues.
35. How often should I commit to github?
  - You should commit every time you want to release a new build after implementing a new features, also make sure that everything works before hand and nothing breaks.
36. What is the command we would use to push our repo up to github?
  - git push -u origin branch-name
37. Walk me through Lambda's git flow.
  1. Create your own repo by forking teh specified project.
  2. Make sure that your forked repo has your name, if not then it's not a fork.
  3. Get the repo url by clicking the GitHub's clone or download button. You can also copy the url from your browser.
  4. You want to then cd into a directory of your choice in your computer.
  5. You want to use the command git clone <lin-to-repo> in order to clone the repo to your desired directory.
  6. Also make sure to cd into the newly cloned directory in your computer.
  7. Create a new branch by using the command git checkout -b 'branch-name'.
  8. Start to work on your repo and get it done.
  9. When done, you want to use the command git add . and git commit -m 'branch-name' to get your branch ready.
  10. After all of those command, finally push your repo to GitHub using the command git push -u origin 'branch-name'.
  11. Your project at this point should be up on GitHub with the new branch and you can submit a PR (Pull Request).
  12. Go to your branches and change it to the one that you created earlier, then press the compare & pull request button.
  13. You should be on a different section now, where there should be a button that says create pull request. You are more than welcome to leave an additional comment and change the title.
  14. Finally click the create pull request button and you are all set to go.

Stretch Questions

1. What is the difference between an inline element and a block element?
  - An inline block displays content in a line horizontally, a block element creates a new line on each element and makes them display horizontally.
2. What happens when an element is positioned absolutely?
  - When an element is positioned absolutely it will not follow inline or block rules, but instead be placed on top of previous content. You can align it by using styling and passing more properties later.
3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?
  - You can pass { height: 100%; } or { width: 100% } or change those properties to a certain amount of pixels, depending on what you're trying to accomplish.
4. Name 3 elements that are display block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
  - block: <div>, <p>, <nav> | inline: <a>, <img> | inline-block: <button>
5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them?
  - The box model is content from HTML, it then has padding, margin, and border that can be set on CSS. You make elements respect their width by passing padding, margin, and border properties in CSS. If an elements display is set to absolute then it will not respect any of the associated values.
